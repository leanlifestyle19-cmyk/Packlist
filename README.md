# 🎒 PackList

> Smart packing lists for every trip. Never forget your sunscreen again.

---

## What it does

PackList gives you ready-made packing lists for 7 trip types, plus a blank template. Check items off as you pack, track your progress, add custom items and categories, and share your list as text.

**8 built-in templates:**
| Template | Best for |
|---|---|
| 🏖️ Beach | Sun, sea, island trips |
| 💼 Business | Work travel, conferences |
| ⛷️ Ski | Snow sports, mountain trips |
| 🏕️ Camping | Outdoor, hiking, wilderness |
| 🏙️ City Break | Short city trips |
| 🎒 Weekend | Quick 2–3 day getaways |
| 🌴 Tropical | Long-haul warm destinations |
| 📝 Blank | Build your own from scratch |

---

## Key features

- One-tap list creation from templates (pre-filled items by category)
- Check off items as you pack — progress bar updates live
- Add custom items inline per category
- Add custom categories with icon picker
- Filter view: All / Unpacked / Packed
- Pack All / Unpack All quick actions
- Share list as formatted text (WhatsApp, email, etc.)
- Export/import as JSON
- Dark mode, fully offline, installable as PWA

---

## Deploy to GitHub Pages

1. Create a new GitHub repo called `packlist`
2. Upload all 4 files: `index.html`, `manifest.json`, `sw.js`, `README.md`
3. Settings → Pages → Branch: main → Save
4. Live at: `https://YOUR_USERNAME.github.io/packlist/`
